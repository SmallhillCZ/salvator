# Automatické přepisy kázání z Akademické farnosti Praha

Přepisy se generují pomocí [OpenAI Whisper](https://openai.com/research/whisper) na základě záznamů kázání dostupných na webu [Akademické farnosti Praha](https://www.farnostsalvator.cz/kazani) každou hodinu.

Přepis je následně zkontrolován a případně upraven pomocí [OpenAI ChatGPT](https://openai.com/chatgpt).

⚠️ Pozor! Přepisy nemusí odpovídat originálu.
